## How to send (SmoothSales dashboard)

1. Open your SmoothSales app (e.g. the Vercel URL).
2. **Service:** E Lion Music.
3. **Audience:** **A&R / Level Up (labels, management, industry)**.
4. **Template:** Leave as **Initial: E Lion Music – A&R / Level Up…**.
5. **Recipients:** Paste emails (one per line) from `levelup-3.csv`, or import the CSV:
   - Open `levelup-3.csv` in Excel/Sheets, copy the first column (email) and paste into the “Paste emails” box; optionally fill names from column 2 so “Hi {{Name}}” is personalized.
   - Or use only the email column: `email1@domain.com, email2@domain.com, …` (comma or newline separated).
6. Click **Send campaign**.

Sends go through Resend; throttle is ~600 ms per recipient. Check Vercel logs or Resend dashboard for bounces/suppressions.

## CSV format

- **email** – required.
- **name** – used for “Hi {{Name}}” (e.g. “A&R Team”, “Reach Records Team”).
- **nameOfOrganization** – for your reference; API supports it if your UI sends it.

You can paste “email only” and leave names blank; the template will use “there” when name is missing.
